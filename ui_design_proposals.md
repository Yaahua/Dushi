# 《都市余烬》UI 界面细化方案 (v1.0)

## 一、 UI 设计核心原则
1.  **信息密度平衡**：通过空间、颜色区分与边框引导视线。
2.  **无感切换**：手机、背包等功能以“内联展开”或“覆盖层”形式出现。
3.  **动态反馈**：打字机效果、闪烁光标、进度条动画。

## 二、 三大 UI 风格方案

### 方案 A：经典终端黑客风 (Classic Terminal)
*   **特征**：单色调（绿/琥珀），高对比度，大量使用 `[ ]`、`#` 符号。
*   **适用**：强调生存系统的严肃感。

### 方案 B：现代极简美学风 (Modern Minimalist)
*   **特征**：TrueColor 灰色调，圆角框线（Unicode），FIGlet 大字体标题。
*   **适用**：强调都市生活质感与手机系统现代感。

### 方案 C：赛博朋克霓虹风 (Cyberpunk Neon)
*   **特征**：青、粉、紫强烈对比，双线边框，ASCII 噪声背景。
*   **适用**：黑市、犯罪与夜生活氛围。
