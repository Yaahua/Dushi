《都市余烬》完整设计共识 v2.3.1

一、游戏定义

一句话描述：纯文字、单窗口、异步多人的现代都市生存模拟器——全服共享一座城市，玩家作为统一出身的难民新移民，体验现金流焦虑、身份维护与高风险生存。

核心特色：
- 纯文字终端美学：无3D/2D画面，等宽字体+ASCII线条，命令行交互
- 全服同城：所有玩家在同一物理城市，真实位置+距离，离散时间运算（City Tick）
- 异步多人：离线策略托管，上线阅读"期间简报"，通过实物载体（便条、痕迹）与受限手机进行弱互动
- 可选进阶教育：基础生存强制教学，大学为高阶可选机构（非强制），项目制认证推动社会流动
- 经济现实主义：现金流账期制（发薪日+账单日），无自动扣费，真实饥饿与驱逐
- 高门槛犯罪：杀人需完整预备链（武器+侦查+三要素），成功率非100%，严惩不贷
- 受限数字生存：手机为基础工具（无硬件焦虑），但无即时通讯，强制面对面社交优先

二、核心体验架构

1. 界面系统（单窗口三区域）

```
┌─────────────────────────────────────────┐
│ [≡] 都市余烬  周四/晴  ¥2,450  📱  💌3  [👤] │
├─────────────────────────────────────────┤
│                                         │
│  ▼ 中央叙事流（唯一滚动区）              │
│                                         │
│  【场景】你醒来在公寓里...                │
│                                         │
│  ┌─ 📱 手机屏幕 ── [关闭] ─┐            │
│  │ 🗺地图  🚗叫车  ☎电话簿              │
│  │ 📨短信  📋客软件  🚨紧急呼叫          │
│  └─────────────────────────┘             │
│                                         │
│  【系统】期间简报：你离线7天，打工收入    │
│  ¥1,400，房租已扣，当前状态：健康         │
│                                         │
├─────────────────────────────────────────┤
│ > 输入指令...  [发送]  [快捷▼]            │
└─────────────────────────────────────────┘
```

关键原则：
- 无弹窗、无跳转、不分屏：邮件/背包/手机都是内联展开的临时区块
- 命令即界面：输入 `去便利店`、`应聘`、`打开 手机`、`申请入学` 触发所有功能

2. 时间系统（双层时钟）

层级	在线状态	离线状态	功能	
世界时钟	RT 1天 = GT 7天	照常流逝	城市天气、新闻、NPC状态始终更新	
个人时钟	RT 24分钟 = GT 1天	策略倍速：RT 1天 = GT 7天	离线时角色按预设策略执行7天日程	

离线机制（策略托管制）：
- 触发：下线时必须选择一项策略（去学校/去上班/在家resting/外出求职/黑市蹲点）
- 执行：离线期间游戏按策略每天跑一轮结算（连跑7次），生成聚合期间简报
- 资源耗尽：若中途现金/食物耗尽，自动降级为"在家resting"（饥饿累积）
- 意外事件：策略执行中可能遭遇随机事件，在简报中体现

三、出身与注册（统一难民制）

入境流程：
1. 排队审核：RT 30分钟-2小时等待期
2. 初始资金：500元 现金 + 旧手机（空电话簿）
3. 初始装备：救济包（基础衣物）
4. 限制区域：前 7天（GT）只能在火车站区活动（隔离墙）
5. 基础教学：难民服务中心强制安全简报（基础生存知识测试，防骗/账单/紧急呼叫）
6. 身份激活：领居住证（无职业标签，仅"市民"身份）

防作弊：设备指纹、新移民标签[新]持续30天（犯罪惩罚加倍）

四、角色与生存

1. 角色创建
- 点数：100点分配至四维（体质/神经质/外向/智力），单项上限40点
- 年龄：青年（免疫杀害）/中年/老年（死亡风险递增）

2. 死亡与继承
- 继承：需遗嘱公证（500元+GT 3天），遗产税分级（30%/50%）
- 数字遗产：死者手机可被破解（黑市服务 GT 500元），读取短信/记录

3. 职业与身份（流体身份）
- 无职业选择，只有许可制：通过行为获得标签，需每日维护
- 雇员：离线策略"去上班"可满足排班，旷工当日无薪，连续2天→解雇
- 店主：店铺必须有人看管，关门=亏钱+口碑下降

五、教育系统（分层可选制）

1. 基础生存教学（强制）
- 地点：火车站区难民服务中心
- 内容：入境时7题测试 + 前7天安全简报（租房/账单/110）
- 产出：基础居住证（"市民"身份，非认证标签）

2. 职业教育（可选）
- 地点：各区职业培训中心（地图POI）
- 形式：短期课程（GT 7-14天），项目制考核
- 产出：基础职业许可（店员、司机、日结工等）

3. 大学教育（高阶可选）
- 地点：大学区（独立区块，需审核进入）
- 准入门槛：
  - 持有居住证 GT 30天以上
  - 学费 GT 2,000-5,000元/学期
  - 通过入学考试（基础逻辑测试，防工作室）
- 形式：
  - 学期制（GT 30天=现实约5天）
  - 玩家主动申请入学后才开启课程系统
  - 可中途退学（学费不退）
  - 离线策略新增"去大学"（深度学习模式，需教材）
- 内容来源：
  - 官方课程（60%）：我设定的硬规则（会计/法律/城市管理）
  - 社区课程（40%）：玩家在大学内提交，我审核机制准确性，采纳者获荣誉教授标签+终身免学费+该课程编辑权限
- 产出：
  - 高阶认证标签（会计师、律师、工程师、公务员）
  - 解锁高薪职业（银行职员、市政厅公务员、公司管理层）
  - 知识特权：掌握黑市行情、区域安全评级等信息差（权力来源）

游戏流分支：

```
入境 → 基础测试 → 进入城市（完全自由分支）
              ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
 直接去打工  职业培训   申请大学
（日结/苦力） （店员/司机） （会计/律师/公务员）
    ↓         ↓         ↓
 无学历路线  技校路线   精英路线
（收入低/风险高） （稳定） （高收入+信息特权）
```

核心：玩家完全可以不去大学，通过黑市、体力劳动、开店等方式生存。大学是上升通道之一，非必经之路。

六、手机系统（基础生存工具）

使用限制（状态驱动，无硬件焦虑）：
- 可用：清醒、站立、行走、工作中
- 禁用：睡眠（躺床自动锁屏）、昏迷、被捕（没收手机GT 24小时）、死亡（需继承人破解）
- 信号限制：地铁隧道/地下室无信号（地理限制）

功能：
1. 🗺 地图：实时定位 + 周边玩家密度（模糊），无导航路径
2. ☎ 电话簿：冷启动（空），必须面对面 `交换号码`，上限50个，可填假名
3. 📨 短信：延迟 GT 30分钟2小时送达，70字限制，夜间顺延至 GT 08:00，无群聊
4. 🚗 叫车：发布需求，司机接单后显示ETA，深夜司机可查看乘客背包价值（犯罪诱因），必须线下碰头
5. 📋 客软件：纯信息板（招聘/租房/二手），无聊天功能，无在线支付，所有交易线下完成，评价延迟 GT 3天
6. 🚨 紧急呼叫：110/120/119，一键发送定位。假报警惩罚 GT 7天通讯封禁。120收费 GT 200元，无钱拒载。

七、经济系统（现金流账期制）

双资产结构：
- 现金：随身携带，上限20,000元，可被偷/被抢
- 存款：ATM取现手续费5%，转账延迟 GT 1天到账

账期机制：
- 发薪日：每月15日（GT）工资自动到账（离线也到账）
- 账单日：每月1日（GT）扣房租/水电/话费（¥50/月），余额不足→停水停电→3天后驱逐
- 话费：随房租自动扣，欠费仅保留紧急呼叫

交易限制：
- 单笔现金限额：2,000元（超过必须转账）
- 日累计限额：5,000元
- 新账号保护：前7天（GT）每日现金限额500元

八、杀人机制（高门槛犯罪）

预备阶段（必须完成）：
1. 工具：黑市购买凶器（300-800元，占背包格），徒手成功率0%
2. 侦查：`跟踪` 连续3次以上（GT 3天），或购买黑市通话记录窃取（800-1500元）
3. 三要素：地点（无监控）+ 时间（独处）+ 状态（目标体力<50%）

成功率公式：

```
30%（基础） +10%（有武器） +15%（目标体力<30%） +10%（埋伏） 
-20%（目标有防备） -15%（神经质高）
```

最高约65%，不可能100%。

后果：
- 失败逃脱→通缉（GT 24小时追捕，RT 24小时封禁）
- 成功→"杀人者"标签（全服可见GT 30天），RT 7天硬封禁，GT 30天冷却期

九、城市运转机制

1. 基础设施（三层架构）

层级	控制方	内容	
底层	开发者	城市OS、公立学校（认证中心）、地铁、公立医院、紧急服务、大学区	
中层	过渡	水电/公交特许经营，可IPO，玩家持股51%可接管	
顶层	纯玩家	私营店铺、黑市、灰色地带、客软件信息	

2. 交通与着装

交通：
- 步行：GT 3分钟=RT 15秒，消耗体力
- 地铁：每GT 10分钟一班，隧道内无手机信号
- 打车：手机叫车，深夜有抢劫风险

着装（功能性）：
- 保暖度：冬天户外超GT 30分钟生病
- 整洁度：西装面试+20%，西装搬砖-尊重值
- 隐蔽度：兜帽降低被识别概率

十、社交与交互（异步双轨）

实物层：
- 便条系统：贴在物体表面（冰箱/长椅），可回复/撕掉/无视
- 物品传递：留下物品+"借你伞，记得还"，物品带记忆ID可追溯历任主人
- 环境改造：刻字、涂鸦，永久ASCII痕迹

数字层（受限）：
- 手机短信：延迟异步，无群聊，强制1对1弱连接
- 客软件：纯信息板，无即时聊天

关系可视化：不显示数字好感度，显示空间距离历史（ASCII时间线）
