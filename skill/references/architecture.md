## 游戏定义

纯文字、单窗口、异步多人的现代都市生活模拟器。全服玩家共享同一物理城市，作为新移民从零开始生存。

## 核心架构八大系统

### 1. 时间系统

双层时钟架构：世界时钟（集体节律）与个人时钟（个体流速）。离线期间世界照常运转，角色属性正常消耗（具体消耗规则由制作人后续提供），玩家上线后通过**简报**了解离线期间发生的事情。采用离散运算的 City Tick 机制，支持分区计算与空区块休眠。

### 2. 出身与注册

统一难民制：所有玩家同一起点（火车站），无差别初始资金与装备。区域隔离——新移民受限于初始区域，需满足条件后进入主城。包含防作弊机制。

### 3. 经济系统

双资产结构（随身现金/银行存款），转账延迟到账。账期机制——固定发薪日与账单日，逾期有软性物理后果（停水/驱逐）。交易限制——现金上限与单笔限额，大额申报与延迟审核。

### 4. 生命与死亡

阶段式生命：自选年龄段（青年/中年/老年），决定可被杀与自然死亡概率。自然死亡——长期离线+角色年龄达标后触发失踪死亡。他杀机制——高门槛犯罪（需武器获取、侦查跟踪、三要素满足），非零成功率，严惩罚。继承制度——关系人公证继承，遗产税累进，无人继承则充公拍卖。

### 5. 身份与职业

流体身份：无职业选择界面，通过玩家行为获得标签（学生/雇员/店主）。标签需维护。许可制度——高级职业需前置证书与考试，玩家可自建机构发放许可。

### 6. 城市基础设施

三层架构：底层（开发者控制的公立系统）、中层（可接管的经营性设施）、顶层（纯玩家自建生态）。玩家共建——土地竞拍、建筑建造、公司开设需跑现实流程。物理限制——地块数量有限，交通耗时真实，物品占格与溯源。

### 7. 叙事与事件

三层事件池：环境基础事件、职业专属事件、跨时间连锁事件。触发机制——标签匹配（状态+地点+时间），非纯随机概率。异步交互——实物载体（便条/痕迹/物品传递），玩家可以游戏内碰面随后交换电话。城市日报——AI 抓取异常数据与玩家行为生成，分日报/周报/年报。

### 8. 交互界面

终端叙事：单窗口三区域（状态条、叙事流、命令行）。所有功能模块以内联卡片形式插入叙事流，无弹窗。
空间导航：采用分层拓扑移动。室内层为复合节点（细分客厅/卧室等）；街道层为漫游链（街东段/中段/西段等）。
交互范式：命令即界面。通过点击叙事流超链接或底部命令行输入触发操作。命令行支持基于空间上下文的智能补全。

## 技术栈选型

| 层级 | 技术选型 | 选择理由 |
| :--- | :--- | :--- |
| 后端框架 | Python (FastAPI) | 原生异步 + WebSocket 支持，适合 Tick 运算与实时推送；复杂逻辑处理与 AI 接入优势 |
| 数据库 | PostgreSQL | 复杂 JSON 存储、空间数据计算、强事务处理，适合双资产经济系统 |
| 缓存/队列 | Redis | 高速缓存 + Pub/Sub 机制，契合全服状态同步与异步消息队列 |
| 前端框架 | React + WebSocket | 模块化终端 UI 组件，实时交互反馈（打字机效果、状态闪烁） |
| 部署方案 | Docker + Nginx | 容器化便于环境隔离与扩展，符合渐进式部署路径 |

## 部署路径

渐进部署：静态验证（MVP）→ 全服同城（分区服务器）。运算架构采用乐观锁防冲突、事件队列快照、WebSocket 长连接轻量维护。

## 内容交付分类

制作人按以下五大类提供内容，工程端按需转化为代码：

| 分类 | 包含内容 |
| :--- | :--- |
| 城市规则与数值 | 基础属性定义、经济数值、时间比例 |
| 地理与设施 | 区域描述文本、交通参数、建筑/POI 列表 |
| 社会职业与政策 | 职业系统、教育系统、法律政策、城市动态政策 |
| 角色NPC与社交 | 角色背景、NPC 档案、社交动作逻辑 |
| 交互工具与系统 | 手机 App 功能、实物交互逻辑、犯罪/冲突逻辑 |
